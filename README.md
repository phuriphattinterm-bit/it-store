# it-store
Full-stack IT store app built with NestJS and React

## Instuction

### 1. Clone the repo
```bash
git clone https://github.com/yourname/it-store.git
```

### 2. Start the API — open terminal 1
```bash
cd it-store-api
npm install
npm run start:dev
```

### 3. Start the UI — open terminal 2
```bash
cd it-store-ui
npm install
npm run dev
```
